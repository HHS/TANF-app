Use an Existing Login.gov Account

If you have an existing Login.gov account, follow these steps to get started with the TANF Data Portal (TDP).

  1. Sign into the TANF Data Portal

    Begin here: Visit the TANF Data Portal and select "Sign in with LOGIN.GOV for grantees"

  2. Sign In

    • Enter Email Address
    • Enter Password
    • Click 'Sign in'

  3. Request Access

    Request access to the TANF Data Portal.

    • Enter First Name
    • Enter Last Name
    • In alphabetical order, locate and select your Associated State, Tribe or Territory

  4. Request Submitted

    Your request for access is being reviewed by an OFA admin.

    You will receive an email (example below) when access has been submitted. Please allow up to 1 business day for a response.

  5. Approved for the TANF Data Portal

    Once your request for access has been reviewed and approved. You will receive an email (example below) when access has been granted and to begin working in the TANF Data Portal. Sign in to TDP to begin working in the TANF Data Portal.
